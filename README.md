# SuperCode News App

## Purpose of the website

- User can search for news
- User can select the language of the news
- User can sort the news by popularity, relevance and publication date

## Built with

- Mobile Design
- News API
- Fetch-API
- Grid Layout
- Flexbox Layout
- JavaScript ES6+
